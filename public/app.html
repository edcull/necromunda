    How to Use

    Campaign Configuration

    Arbitrators access all configuration via the ☰ menu (top-right).

    Campaign Configuration — set the campaign name, sector name, number of players (2–8), and number of territories in play.

    Simultaneous Games — toggle on/off and set the batch size (2–8). See the Simultaneous Games section below.

    Territory Configuration — rename any territory or hideout node inline directly on the map, or via the territory detail card.

    Gang Configuration — set each gang's name, icon, colour, house/gang type, and starting stats (Rating, Wealth, Rep, Kills). On mobile, fields stack vertically with labels.

    Trading Post Configuration — override item availability and add custom items (see Trading Post below).

    Delete Campaign — permanently removes the campaign and all its data. This cannot be undone.

    Map & Navigation

    Pan & zoom — drag to pan; pinch or scroll to zoom. On desktop, hold Shift and drag to zoom.

    Minimap — the small HUD (bottom-right) shows the full map. Tap it to jump to any area.

    Nodes — territory nodes are hexagonal; hideout nodes are smaller circles. Tap any node to open its detail card.

    Repositioning (arbitrator) — hold-press a node for ~0.5s until it lifts, then drag it to a new position. Release to drop.

    Node colours — nodes are tinted by their controlling gang's colour. Uncontrolled territories are neutral. A lock icon indicates a pending challenge.

    Gang Legend

    The collapsible legend (bottom-left) lists all gangs ranked by their current sort key. Tap the ◀ arrow to collapse it.

    Sort options — Territories (default), Rep, or Rating. Click the sort buttons to switch.

    Gang names are clickable links that open the gang's Hideout panel directly.

    Each row shows the gang's colour dot, name, and their count for the active sort (territories held, rep, or rating).

    Hideout Panel

    Open by tapping a gang's hideout node on the map, or clicking a gang name in the legend.

    Stat cards — Rating, Wealth, Rep, and Kills are shown as cards. Arbitrators can tap any value to edit it inline.

    Territory boons — all boons from territories the gang controls are listed. Boons that match the gang's house type are highlighted as enhanced boons.

    Spyrer gangs — instead of territory boons, Spyrers show a Terror Level stat and their Base Camp boon.

    Territory list — controlled territories are listed with a link back to their map node.

    Cycle Panel

    Open via the Cycle button (top-left of map). Tracks the full Dominion Campaign — 7 phases total:

    • Occupation ×3 — gangs fight for unclaimed territories
    • Downtime — gangs recover, trade, and regroup
    • Takeover ×3 — gangs challenge each other for owned territories

    Navigate — use the ◀ ▶ arrows to view previous cycles. Only the active cycle allows new challenges.

    Advance (arbitrator) — once ready, click Advance to move to the next phase. The button becomes Roll Back when viewing a past cycle.

    Challenges

    Arbitrators add challenges via + New Challenge in the active cycle panel.

    Challenger — the gang issuing the challenge. Each gang may only challenge once per cycle (until all have challenged, after which additional games are unrestricted).

    Challenged Gang — the defending gang. A gang that was challenged in a previous set must issue their own challenge before they can be challenged again.

    Staked Territory options:

    • Normal — a territory controlled by the defender is put at stake; winner claims it
    • Rescue Mission — always available; used when the challenger has a captured fighter to rescue; no territory changes hands
    • Special Scenario — appears when the defender holds no territories; no ownership change

    Occupation phase — staked territory must be unclaimed. The defender is optional (leave blank for an unclaimed node).

    Recording results — click the result button on a pending challenge card to open the result screen. Select Win (challenger or defender), Draw, or Refused. Update Rating, Wealth, Rep, and Kills for both gangs. Territory ownership updates automatically.

    Deleting — pending challenges can be deleted (trash icon) in the active cycle only.

    Simultaneous Games

    Enable in Campaign Configuration → Simultaneous Games. Set the batch size (2–8 games per set).

    When active, challenges are grouped into numbered sets (Set 1, Set 2, …). All games in a set are played at the same time before the next set begins.

    Constraints within a set:

    • A gang already in the current set (as challenger or challenged) cannot appear in another challenge in the same set
    • A gang challenged in a previous set who hasn't yet issued their own challenge must be a challenger — they cannot be challenged again until they've done so

    Additional games — once every gang has issued their challenge this cycle, all constraints are lifted and any gang may challenge any other freely.

    Challenge cards are grouped by set with divider labels. An Additional Games section appears below.

    Trading Post

    Access via the ⚖ Trading Post button in the top bar.

    Browse tab — full Dominion Campaign trading post catalogue organised by category: weapons, armour, wargear, and ammunition. Each item shows its cost and rarity.

    Seek Rare / Illegal tab — roll for availability of rare or illegal items. Set your base dice, modifiers, and gang type to filter relevant items. The tracker shows which items become available at each roll threshold.

    Arbitrator overrides — arbitrators can change any item's availability level (Common / Rare / Illegal / Unavailable) and add fully custom items with a name, cost, and availability.

    Triumphs

    Access via the 🏆 button (top-left of map, next to the Cycle button).

    Five triumph cards are auto-calculated live from campaign state. The top 3 gangs for each triumph are displayed:

    • Dominator — most territories controlled at campaign end
    • Slaughterer — most enemy fighters taken Out of Action
    • Creditor — highest Wealth at campaign end
    • Warmonger — most battles fought across the whole campaign
    • Powerbroker — highest Reputation at campaign end

    Triumphs update in real time as challenges are recorded and stats change.
